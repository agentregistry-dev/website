Give an agent access to prompts that are published in agentregistry. Prompts are added to the `prompts_loader.py` definition. 

## Before you begin

1. Follow the [Get started](/docs/quickstart/) guide to set up agentregistry and start the agentregistry daemon.
2. [Create an agent](/docs/agents/create/).
3. [Create a prompt](/docs/prompts/create/) and [publish it](/docs/prompts/publish/) to agentregistry.
