Remove an agent that is deployed. Use flags for non-interactive setup or run without flags to open the wizard.

## Usage

```sh
arctl agent remove [name] [args...] [flags]
```

Example: 
```sh
arctl agent remove my-agent
```

## Command-specific flags

No command-specific flags.

## Global flags
```sh
-v, --verbose: Enable verbose output.
-h, --help: Display help information for the command.
```
