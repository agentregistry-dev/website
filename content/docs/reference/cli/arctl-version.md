Displays the version of `arctl`.

## Usage

```sh
arctl version [flags]
```

Examples:
```sh
arctl version
arctl version --json
```

## Command-specific flags

```sh
--json   Output version information in JSON format
```

## Global flags
```sh
-h, --help: Display help information for the command.
```
